# Todo App

A modern, fully-featured TODO application built **entirely without writing a single line of code manually**.

## How It Was Built

This entire application was created using **only Claude Code CLI commands**. No manual coding was involved - every line of code, configuration, and setup was generated through natural language conversations with Claude.

### The Process
1. Described the desired features in plain English
2. Claude Code generated all components, hooks, and configurations
3. Deployed to Vercel with a single command

## Features

- Create, edit, and delete todos
- Mark todos as complete with checkbox
- Inline editing (double-click to edit)
- Delete confirmation modal
- Data persists in localStorage
- Dark/light mode toggle (respects system preference)
- Smooth animations with Framer Motion
- Fully responsive design

## Tech Stack

| Technology | Purpose |
|------------|---------|
| React 18 | UI Framework |
| Vite | Build Tool |
| shadcn/ui | UI Components |
| Tailwind CSS v4 | Styling |
| Framer Motion | Animations |
